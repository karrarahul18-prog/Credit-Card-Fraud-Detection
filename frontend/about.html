How FraudShield Works

A simple guide to understanding how our AI-powered Credit Card Fraud Detection system works — built specifically for Indian transactions.

📋 Step-by-Step Process

1

Enter Transaction Details

You fill in the details of the transaction: amount in ₹, time, merchant, city, and whether it's online or international.

2

Flask Backend Receives Data

The frontend sends the data to our Python Flask server via a REST API (POST /predict).

3

ML Model Analyzes the Transaction

A pre-trained Random Forest Classifier processes 11 features and predicts whether the transaction is fraud or normal.

4

Result + Explanation Shown

You get a clear verdict (Fraud / Safe), fraud probability %, risk level (High/Medium/Low), and reasons why it was flagged.
